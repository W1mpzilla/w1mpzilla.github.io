Web Features Embedded Into The Page?

A quick list of the few Web design features embedded into the page:


  • Responsive

    Responsive Web Design is about using HTML and CSS to automatically resize, hide, shrink, or enlarge, a website, to make it look good on all devices from desktops to tablets and smartphones.
    This page will resize automatically if the browser shrunk or if viewed on a smartphone.


  • Additional CSS sheet

    Developers often override or customize default Bootstrap CSS styles when working on web development projects. Most developers do it to make changes to Bootstrap fonts, borders, colors, or other Bootstrap styles.
    In this page the font white shadow is applied from a CSS sheet on top of the Bootstrap styles.


  • Scripts

    A dynamic page can send and receive up to date information that was not available when the web page was created.
    Scripts can be viewed as small computer programs written in a variety of scripting languages like JavaScript, VBScript, Perl, PHP, ASP, updating dynamically the page information. In this page the Translate form, the date, the time, the font type, etc, come from scripts.
